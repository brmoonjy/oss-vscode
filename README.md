# 2. LINUX
> _The following is based on Ubuntu_
## 2.1. COMMAND TYPE
1. File Management : `ls`, `cd`, `mkdir`, `rmdir`, `touch`, `vi`, `cat`, `mv`, `rm`, `find`
2. Shell : `chsh`, `echo`, `export`, `alias`, `unalias`, `history`
3. File Access Management : `chmod`, `umask`, `chown`
4. Pakage Management :
   | Function | apt Type | apt-get / apt-cache Type|
   |:---:|:---:|:---:|
   | Pakage Search | apt search | apt-cache search |
   
5. Archive/Compression : `tar`, `gzip`, `gunzip`, `bzip2`, `bunzip2`
6. Process : `ps`, `pgrep`, `kill`, `sleep`, `jobs`, `fg`, `bg`
7. System User Management 
   - User : `useradd`
   - Group : `groupadd`
8. File System Management : `fdisk`
9. Network Management :`nmcli`
